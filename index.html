Design Principles

Zero trust access

Access granted on identity and policy, never network location.

No public exposure

Zero inbound ports opened on the edge network.

Minimal attack surface

Services only reachable through the overlay network.

Policy segmentation

Tag-based ACLs restrict access at the service level.

Tag-based ACL policies applied for segmentation
Remote access validated from external networks
Operational Notes
Tailscale auth state is independent of system uptime — re-auth on reboot is expected behavior.
Subnet routing requires both route advertisement on the router and acceptance on each client node.
NAT is required when upstream routing has no awareness of the overlay network space.
Tag consistency must be maintained when re-authenticating nodes — drift breaks ACL enforcement.
